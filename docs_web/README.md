# SKN28-3rd-1Team Docs Web

프로젝트 소개, 기술 스택, 프론트엔드/백엔드/RAG 작업 방향을 한 화면에서 확인하는 문서 웹입니다.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

GitHub Pages 배포를 위해 `vite.config.ts`의 `base`는 `/SKN28-3rd-1Team/`로 설정되어 있습니다.
